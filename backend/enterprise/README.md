# Enterprise Features

This folder contains placeholder stubs for enterprise features.

## Community Edition

These features are not available in the community edition. The stub files exist for:
- Import compatibility
- Type checking
- IDE autocomplete

The routers from these stubs are NOT registered in community edition.

## Enterprise Edition

The enterprise edition is hosted by KenisLabs on their cloud infrastructure with:
- **Azure AD / Entra ID SSO**: Enterprise single sign-on
- **Priority support**: Direct access to the KenisLabs team
- **Future features**: Teams, audit logs, advanced security, and more
